<objective>
Translate the 8,192-word English medium list into Standard Malay (DBP orthography), run the full pipeline (normalize → dedup → Schlinkert prune → verify → whittle-to-target → format), and create the SQLite database — producing `jalan-dewan-bahasa-sederhana.txt` and `db/jalan-dewan-bahasa-sederhana.db`.

**Purpose:** The medium list (8,192 words) is the first large-scale test of the pipeline, 6.3× larger than the alpha list from Phase 2. Success here proves that: (1) Schlinkert pruning scales to moderate list sizes (~8K words) with Malay agglutinative morphology, (2) the word supplementation + whittle-to-target workflow from Phase 2 works at scale, (3) entropy target of ~13 bits/word is achievable, and (4) DBP orthography quality holds at this vocabulary size.

**Output:** 1 wordlist `.txt` file + 1 SQLite `.db` file + scratch intermediates. No dice variant (confirmed per architecture docs — medium list has no dice variant).
</objective>
