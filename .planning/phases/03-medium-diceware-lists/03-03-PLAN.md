These patterns catch the most common Indonesian spelling markers found in the Phase 2 research:

```bash
INDONESIAN_PATTERNS=(
  # Suffix differences (most important: -itas → -iti)
  'itas$'
  # Specific word patterns (word-boundary)
  '\bkarena\b'
  '\buniversitas\b'
  '\bnopember\b'
  '\bagustus\b'
  '\baktifitas\b'
  '\bfasilitas\b'
  '\bsistim\b'
  '\bpropinsi\b'
  '\bkwalitas\b'
  '\bbisa(\b|[^s])'  # "bisa" as "can" (Indonesian) — but NOT "bisa" meaning "venom"
  # -sih suffix (but exclude false positives)
  'sih$'
)
```

**Step 2 — Run the audit on medium list:**
```bash
echo "=== DBP Orthography Audit ===" > scratch/dbp-audit-sederhana.txt

echo "" >> scratch/dbp-audit-sederhana.txt

for pattern in "${INDONESIAN_PATTERNS[@]}"; do
```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
