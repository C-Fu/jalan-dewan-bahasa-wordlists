Create a single, self-contained HTML file at `html/index.html` with:

**Functionality:**
1. Load sql.js WebAssembly SQLite engine from CDN
2. Fetch `../db/jalan-dewan-bahasa-kecil-qwerty.db` as binary (ArrayBuffer)
3. Initialize SQLite database, query random words: `SELECT word FROM words ORDER BY RANDOM() LIMIT N`
4. "Generate" button produces a passphrase of configurable word count (default 6, range 3-12)
5. Clicking a generated word replaces it with a new random one from the list
6. One-click copy button to copy passphrase to clipboard
7. Display the entropy estimate (bits) for the generated passphrase

**Styling (per DESIGN.md):**
- Dark background: gradient from #101015 to #1A1A2E
- Accent color: #6366F1 for interactive elements, borders, glows
- Glass card surface: background rgba(255,255,255,0.03), backdrop-filter blur(12px), border 1px solid rgba(255,255,255,0.06), border-radius 12px
- Outer gradient border shell around the main card (gradient hairline from rgba(255,255,255,0.2) to transparent)
- Primary button: background #FFFFFF, color #000000, border-radius 8px, padding 8px 20px, font-weight 500
- Secondary button: ghost style with accent border
- Headings: system font, white, light weight
- Body text: system font, #A3A3A3 or white depending on context
- Word pills: each word in a pill/chip with subtle background and accent border
- Entropy display: subtle secondary text
- Copy feedback: brief tooltip/success state on button
- Responsive: works on mobile (stack vertically) and desktop (centered card, max-width ~600px)
- Card padding: 32px (matching DESIGN.md card padding)
- Section padding: 24px

**Implementation notes:**
- Load sql.js via CDN: `https://cdn.jsdelivr.net/npm/sql.js@1.10/dist/sql-wasm.wasm` (sql-wasm.js)
- Use `initSqlJs()` to initialize, then `new SQL.Database(new Uint8Array(buffer))` to load
- Fetch the .db file from a relative path so it works when served from any static server
- Include error state if database fails to load
- All text labels in English or Malay (user's choice — use Malay for UI labels since this is a Malay wordlist project)
