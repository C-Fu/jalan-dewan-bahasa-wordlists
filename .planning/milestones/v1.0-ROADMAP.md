# Milestone v1.0: Jalan Dewan Bahasa Wordlists

**Status:** ✅ SHIPPED 2026-07-01
**Phases:** 1-4
**Total Plans:** 10

## Overview

Direct translation of the Orchard Street Wordlists (7 English passphrase wordlists) into Standard Malay using a semi-automated pipeline: DeepSeek v4 Pro translation → dedup → normalization (Tidy) → Sardinas–Patterson verification → Schlinkert pruning (Tidy -K) → final verification (Tidy -AAAA). Each wordlist is also published as a SQLite database (db/*.db) for programmatic use. Build smallest lists first to validate the pipeline before scaling to the largest (17,576 words).

## Milestone Summary

### Key Decisions
- **Pipeline order**: Translation (DeepSeek v4 Pro) → Dedup → Normalize (Tidy) → Schlinkert Prune (Tidy -K) → Verify (Tidy -AAAA)
- **Build order**: alpha → qwerty → medium → diceware → long (smallest-first de-risking)
- **Malay orthography**: DBP Standard Malaysia (not Indonesian/KBBI)
- **No cross-list merging**: Each list translated and pruned independently to maintain unique decodability
- **Supplementation strategy**: Multi-round supplement-and-reprune for large lists where translation yields insufficient unique words
- **Profanity/DBP**: Reuse Phase 2 curated 33-entry profanity reject list across all phases
